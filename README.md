# AI Concepts Lab Report

## Overview

This repository contains practical machine learning and deep learning lab exercises, including linear regression, logistic regression, and PyTorch basics. The goal is to demonstrate key concepts through code examples, visualizations, and hands-on experimentation.

- `Linear Regression/` - Simple linear regression case study using salary data.

## Pytorch Basics Lab

### Objective

Introduce PyTorch fundamentals and provide a clear comparison between scalars, vectors, and tensors. Demonstrate tensor creation, arithmetic operations, reshaping, and simple image loading.

### Key Concepts

- **Scalar**: a single number with magnitude only.
- **Vector**: an ordered list of numbers with magnitude and direction.
- **Tensor**: a multi-dimensional array that generalizes scalars and vectors.
- **Tensor operations**: addition, subtraction, multiplication, matrix multiplication, and reshaping.
- **Image handling**: loading and displaying images using PIL and Matplotlib.

### Notebook Highlights

- Created tensors using `torch.tensor`, `torch.rand`, `torch.zeros`, `torch.ones`, `torch.eye`, and `torch.arange`.
- Compared element-wise multiplication with matrix multiplication.
- Used `reshape` and `view` to change tensor dimensions.
- Loaded and displayed an image from a remote URL using `requests`, `PIL.Image`, and Matplotlib.
